# STPRSPA - Sitio Web Institucional

Web oficial del **Sindicato de Trabajadores de la Prevención de Riesgos Laborales, Salud Ocupacional y Protección Ambiental**.

## 🛠️ Tecnologías
- **React** + **Vite**
- **Tailwind CSS** (Diseño Mobile First)
- **Formspree** (Gestión de afiliaciones)

## 🚀 Características
- Landing page con enfoque en la autoridad gremial.
- Sistema de afiliación técnica con formulario integrado.
- Diseño optimizado para la cuenca industrial (Neuquén y Río Negro).
